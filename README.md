# Finam Bot

Бот для автоматических встречных ордеров на Finam.

## Установка

```bash
cd finam-bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Запуск

```bash
source venv/bin/activate
FINAM_TOKEN=твой_токен python bot.py
```

## Пример

```bash
FINAM_TOKEN=eyJraWQi... python bot.py
```
